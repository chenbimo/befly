# Internal 目录说明

⚠️ **重要提示**

此目录下的所有文件由 **befly-admin** 框架管理，请勿修改！

## 自动更新

运行以下命令会自动更新此目录：

```bash
befly sync:admin
```

## 注意事项

1. ❌ **请勿修改** internal 目录下的任何文件
2. ❌ **请勿删除** internal 目录
3. ✅ 如需自定义插件，请在 internal 目录外创建文件，并使用不同的导出名称

## 保留的导出名称

框架保留以下导出名称，请勿在自定义插件中使用：

- `$Http` - HTTP 请求封装
- `$Storage` - 本地存储封装
- `router` - Vue Router 实例
- `useAppStore` - 应用状态管理

## 文件用途

此目录包含框架核心插件，确保管理系统的基础功能正常运行。
