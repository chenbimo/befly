# Internal 目录说明

⚠️ **重要提示**

此目录下的所有文件由 **befly-admin** 框架管理，请勿修改！

## 自动更新

运行以下命令会自动更新此目录：

```bash
befly sync:admin
```

## 注意事项

1. ❌ **请勿修改** internal 目录下的任何文件
2. ❌ **请勿删除** internal 目录
3. ✅ 如需创建业务页面，请在 internal 目录外创建文件

## 文件用途

此目录包含框架提供的管理页面（用户管理、角色管理、菜单管理等），确保管理系统的基础功能正常运行。
